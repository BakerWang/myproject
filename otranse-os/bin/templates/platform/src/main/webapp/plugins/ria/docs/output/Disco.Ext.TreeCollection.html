Class Disco.Ext.TreeCollection

Package:Disco
Defined In:core-3.2.js
Class:Ext.TreeCollection
Extends:Ext.util.MixedCollection
树状节点数据存储器，该缓存器配合Disco.Ext.TreeNodeUtil，提供了快速查询树节点及内在节点内容变更事件响应等相关方法
如果在应用中如果需要缓存树状结构的数据，可以使用该缓存器。
