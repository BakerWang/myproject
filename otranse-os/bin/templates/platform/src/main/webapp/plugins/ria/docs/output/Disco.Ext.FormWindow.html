Class Disco.Ext.FormWindow

Package:Disco
Defined In:core-3.2.js
Class:Ext.FormWindow
Extends:Window
扩展的包含表单的窗口组件。
主要的扩展是在窗口关闭的时候，去检查表单是否已经做了修改，如果修改了，可以提示是否保存表单内容。
要使用该组件，必须保证该窗口内的第一个组件就是form表单。及window.getComponent(0)就是formPanel.
