    3天前，家住新都的刘女士2岁女儿在家里玩耍爸爸的弹珠枪时，不慎拨动了扳机，大量弹珠打进了刘女士的后脑，医生在X射线下紧急为她做了近3个小时的手术，取出35颗铁质弹珠（成都商报昨天曾报道）. 成都商报的这篇报道也引起了新都警方的注意，昨日上午，新都警方根据新闻中提供的相关信息，挡获了刘女士的丈夫张东强（化名），并缴获了这把弹珠枪。据张东强交代，这把枪是他网购装修用的射钉枪改装而成，用于自己打鸟娱乐。警方表示，将把这把枪送到专业鉴定机构去检验，如果被鉴定为“枪械”，张东强将被依法追究法律责任。 看商报 刑警发现涉枪线索“这次警方能这么迅速地掌握这桩涉枪伤人案的案情，并及时把嫌疑人和枪找到，消除社会安全隐患，真的要感谢商报的新闻报道，”昨天下午，成都市公安局新都区分局刑警大队技术室副主任何维再三表示了对成都商报的感谢。 据何维介绍，昨天上午刚上班，刑警大队一位民警在浏览成都商报时，立即就被这篇“2岁娃娃玩枪误伤母亲”的新闻引起注意。“有很大可能是一件涉枪案件！”刑警们在短暂讨论后得出了一致结论。按照新闻中提到线索，刑警立即出发前往医院了解情况。在刘女士的病房，刑警正好遇到陪在妻子病床边的张东强，他被带回刑警大队接受调查。 随后，办案民警在张东强家里找到了这把枪、剩余的4发空爆弹和数十粒铁砂丸。整个枪身长达1.4米，发射部分是用一把装修用的弹簧射钉枪改装而成，两根口径5毫米的钢管焊接在射钉枪上充当枪筒。“这种自制土枪的发射命中率和火力都不小，非常危险。” 何维说。 害人害己 小伙或面临法律制裁 办案民警透露，这把枪子弹上了膛、装了铁砂丸后不能退出来，张东强平时就把枪竖着靠在门后。事发时他不在场，听到枪声后才冲进妻子和女儿所在的房间里，发现枪已经倒在地上，而女儿坐在离枪很近的地上，很可能是女儿好奇去摸枪，并不慎碰到了扳机，引发了枪响。 “通过走访，我们发现嫌疑人和妻子感情不错，暂时没有发现他有主观犯罪意图。”何维表示，目前案件还在进一步侦查当中，这把自制枪将被送到专业鉴定机构进行检测，一旦被认定是具有一定杀伤力的“枪械”，张东强将面临因非法持有枪支而被刑事拘留，并遭到起诉。成都商报记者 王欢 摄影报道
