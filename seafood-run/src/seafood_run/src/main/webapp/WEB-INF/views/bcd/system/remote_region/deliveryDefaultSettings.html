友情提示

  • 选择完子地区确认后，系统并未保存，需要点击页面底部的保存按钮系统才会保存设置的地区
    • 默认单品非偏远地区包邮价
    • 注：每个订单可能包含多个商品，当订单中该商品的购买总价超过设置值，则该商品包邮
    • 默认单品非偏远地区邮费
    • 注：当订单中只存在一种商品且该商品的购买总价低于设置值，则该商品应付设置的邮费；当订单中含有多种商品且多种商品的购买总价低于设置值，取不包邮商品中邮费最高的值；
    • 默认偏远地区邮费
    • 注：当订单不满足包邮规则，且收货地址为偏远地区，则默认邮费为该设置值
