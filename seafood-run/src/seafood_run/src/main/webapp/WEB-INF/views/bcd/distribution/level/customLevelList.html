友情提示

  • 开启微店等级管理功能后，系统默认有一个等级，无法删除。
  • 系统等级由用户自主设置，按照佣金金额来判断等级条件。请根据实际情况合理设置等级条件。
  • 设置微店等级后，可以按照等级设置佣金比例。
  • 默认等级的微店等级佣金按照基本佣金结算。其他等级佣金为基本佣金×倍数。（包含自定义佣金的商品）请具体核算成本后设置佣金。
