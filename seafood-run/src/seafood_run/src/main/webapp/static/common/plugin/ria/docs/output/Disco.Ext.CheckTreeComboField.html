Class Disco.Ext.CheckTreeComboField

Package:Disco
Defined In:core-3.2.js
Class:Ext.CheckTreeComboField
Extends:Ext.TreeComboField
xtype:checktreecombo
树状下拉框,可以下拉出一个TreePanel并且支持check选择
该组件的使用方式和TreeComboField基本相同。
该组件的一些特性：1，在选中了子节点后，父节点也能跟着选定，但不放入选中列表。2，选中了父节点，该父节点下的子节点自动全部选中。 
new Disco.Ext.CheckTreeComboField({
fieldLabel : '部门',
name : 'dept',
tree : new Ext.tree.TreePanel({
border : false,
autoScroll : true,
rootVisible : false,
loader : new Ext.tree.TreeLoader({
url : 'data/tree.json',
baseAttrs : {
checked : false
}
}),
root : new Ext.tree.AsyncTreeNode({
id : 'id',
text : '部门'
})
})
})
