商城用户操作手册

海鲜之家商城是海鲜之家基于微信界面制作的，专注于原产地美食的吃货分享平台。您在微信公众号中搜索“海鲜之家商城”，就可以看到我们的海鲜之家商城公众号。

进入公众号之后，您可以看到我们商城的对话框界面，选择“进入商城”，就可以进去我们海鲜之家的商城了。

这就是我们的商城的主界面了，您可以看到我们出售的产品，目前有微冻系列、海洋休闲食品、阳澄湖大闸蟹三大类产品。您就可以直接点击上面的产品进行购买了。

二、客户如何成为海鲜之家的分销商？

如果您对我们海鲜之家的产品满意，同时您又有兴趣将海鲜之家的产品分享给您的朋友，利用空闲时间来赚取佣金，那么您就可以申请成为海鲜之家的分销商。

您的申请提交之后，我们后台会对您的资格进行审核，审核通过之后公众号会第一时间给您发送消息，您就是我们的分销商了！

三、如何成为海鲜之家的一级分销商？

①是海鲜之家连锁的实体加盟店（目前一级分销商只对加盟店开发）
②在海鲜之家商城购买体验产品满1000元
满足以上两个条件之后，您有兴趣将海鲜之家的产品分享给您的朋友，那么您就可以申请成为海鲜之家的一级分销商。一级分销商可以赚取产品的差额。

您的申请提交之后，我们后台会对您的资格进行审核，审核通过之后，公众号会第一时间给您发送消息，您就是我们的销售商了！
